# AQI Data Analysis and Regression Modeling
Air Quality Index analysis using regression modeling, PCA, LASSO regression, and feature selection techniques.

## Overview

This project analyzes Air Quality Index (AQI) data to identify factors influencing air pollution levels and evaluate predictive models using statistical and machine learning techniques.

## Objectives

- Explore AQI patterns and relationships among variables
- Identify significant predictors of air quality
- Compare feature selection and dimensionality reduction approaches
- Develop interpretable predictive models

## Dataset

- Air Quality Index (AQI) dataset
- Focused analysis on Delhi observations

## Techniques Used

- Exploratory Data Analysis (EDA)
- Multiple Linear Regression
- LASSO Regression
- Principal Component Analysis (PCA)
- Subset Selection
- Model Evaluation

## Tools

- R
- Jupyter Notebook
- Excel

## Key Outcomes

- Performed feature selection using LASSO Regression
- Applied PCA for dimensionality reduction
- Evaluated regression models and predictor significance
- Generated visualizations for data exploration and model interpretation

- Statistical Modeling
- Data Analysis
- Regression Analysis
- Feature Selection
- Data Visualization
- Model Interpretation
